проэкт The Loop

Этот сайт является проектом для тренровки использования HTML тэгов и стилей SASS . Он состоит из 1 страници с несколькими блоками с тектсом и картинками,а так же определённым стилем текста. Особенностью работы с данным проктом было использование кампилятора Preprose для проебразования SASS в CSS. Данный проект создавался для доп задания на модуль по потнаправлению Веб-Информатика.
