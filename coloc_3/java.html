проэкт EFS.kg

Это курсовой проект написанный во время обучения на 1-ом курсе в КГИПИ. За основу взят язык программирования JAVA и фреймворк Spring, а так же база данных MSQL. Проект EFC.kg представляет собой платформу для подачи заявки на обучение в школу и детские сады города Бишкек. Для просмотра кода нажмите сюда
